Team Project:
This project is an emotion analysis tool where you paste any text and instantly get a breakdown of its emotional tone — displayed as a radar chart, color-coded sentence highlights, and a plain-language verdict like "this sounds mostly annoyed with a hint of disappointment." It's trained on Google's GoEmotions dataset (58k Reddit comments across 27 emotions) and compares classical ML models like Logistic Regression and SVM against a fine-tuned DistilBERT transformer. The core use case is a "second opinion before you hit send," helping users catch unintended tone when writing while tired or frustrated.

Team Project:
TacTool is a VR project for the Meta Quest 3 that lets users interact with virtual objects using tools like hammers, chisels, and files, each triggering distinct haptic vibration patterns to simulate material properties such as hardness, texture, and friction. Built in Unity with 3D assets from Blender and haptic patterns designed in Meta Haptics Studio, the system generates 12 unique vibrotactile signatures by varying vibration intensity, duration, and temporal envelope. The project concludes with a user study evaluating how well participants can identify materials through different combinations of haptic, visual, and audio feedback.
